### 配布側（あなた）とインストール側（相手）の手順
配布側（あなた）の作業
1. npm run build:win でビルド
2. dist\NoiseRide Setup 1.0.0.exe を相手に渡す（ファイル送信・共有フォルダなど）

インストール側（相手）の作業
1. NoiseRide Setup 1.0.0.exe を受け取り、ダブルクリック
2. ウィザードが開くのでそのまま「次へ」→「インストール」
3. 完了するとデスクトップとスタートメニューに「NoiseRide」が登録される
以降はショートカットをダブルクリックするだけで起動できる
相手に必要なもの: インストーラー EXE ファイルのみ（Node.js・npm・開発環境は一切不要）

バージョンアップ時
1. あなたが package.json の version を変更してビルド
2. 新しい NoiseRide Setup X.X.X.exe を相手に渡す
3. 相手は旧バージョンをアンインストールせずそのまま実行 → ウィザードがアップグレードを検出して更新完了MP3 登録情報・キーバインド設定はそのまま引き継がれる
