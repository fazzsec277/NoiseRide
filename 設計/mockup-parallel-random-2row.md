```
   アクティブ  通常        通常       通常
```

### ランダム複数ON時（BGM・SE がランダム再生中）

```
┌──────────────────────────────────────────────────────────────────┐
│ ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐                         │
│ │ 全体 │  │🔶BGM │  │🔶 SE │  │アンビ│  +                       │  全停止■  KB:ON  +追加
│ │· ⇄ ·│  │« ⇄ »│  │« ⇄ »│  │· ⇄ ·│                         │
│ └──────┘  └──────┘  └──────┘  └──────┘                         │

   通常       ランダムON  ランダムON  通常
             （オレンジ）（オレンジ）
  ※ 🔶 = タブ背景がオレンジ系にハイライト
```

## タブ各状態の詳細

### 通常（ランダムOFF）
```
┌──────┐
│ BGM  │   ← プリセット名（通常色）
│· ⇄ ·│   ← « » はグレーアウト、⇄ はクリック可
└──────┘
```

### ランダムON（再生中）
```
╔══════╗
║🔶BGM ║   ← プリセット名（オレンジ）+ タブ枠オレンジ
║« ⇄ »║   ← «»⇄ すべてオレンジ色、クリック可
╚══════╝
```

### アクティブタブ（選択中）＋ランダムOFF
```
┌──────┐
│▌全体 │   ← 左に紫のアクセントライン or タブ背景が紫系
│· ⇄ ·│
└──────┘
```

### アクティブタブ（選択中）＋ランダムON
```
╔══════╗
║▌BGM  ║   ← 紫アクセント + オレンジ背景の複合
║« ⇄ »║
╚══════╝
```

## サイズ感

| 要素 | Before（1段） | After（2段） |
|------|-------------|-------------|
| タブ幅 | 最大 ~180px | ~60〜80px（名前次第） |
| タブ高 | 30px | ~44px（TabBar全体を +14px 拡張） |
| TabBar高 | 44px | ~58px |

タブ幅は大幅に削減し、代わりにTabBarの高さを少し増やす。

---

## ボタン仕様

| ボタン | サイズ | 無効時 | 有効時 |
|--------|--------|--------|--------|
| `«`（前曲） | 18×18px | グレーアウト（opacity 0.25）| オレンジ |
| `⇄`（ランダム切替） | 24×18px | グレー | ON時オレンジ |
| `»`（次曲） | 18×18px | グレーアウト（opacity 0.25）| オレンジ |

- `«` は再生履歴がある場合のみ有効
- `»` はランダムON時は常に有効
- `⇄` は常にクリック可能（ON/OFFトグル）
